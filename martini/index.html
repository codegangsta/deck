m.Get("/", func() (int, string) {
  return 418, "i'm a teapot" // HTTP 418 i'm a teapot

m.Get("/", func(res http.ResponseWriter, req *http.Request) {
  res.Write([]byte("Hello world!")) // 200 Hello world!

Chapter 5:

Middlewhat?

Middlewares are just Handlers


m.Use(func() {
  // do middlewary things
})

Middleware is executed in the order that they are declared.


func Auth(res http.ResponseWriter, req *http.Request) {
  if req.Header.Get("X-API-KEY") != "secret123" {
    res.WriteHeader(401)
  }
}
        

m.Use(Auth)

Yup. That easy

Extreme Reuse

Next

Yield to the rest of the stack

Totally optional


func Log(log *log.Logger, c martini.Context) {
  log.Println("Before")

  c.Next() // run the rest of the stack

  log.Println("After")
}

Unlike other frameworks, Next is completely optional.
